# libft_games

Jogos educativos para aprender as funções do projeto libft da 42 School.

## Como rodar localmente

**Requisitos:** Node.js 18+ e npm (ou pnpm)

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev
```

Acesse http://localhost:5173

## Build para produção

```bash
npm run build
npm run preview
```

## Jogos disponíveis

- **Quiz de Funções** — 10 perguntas aleatórias sobre ft_*
- **Jogo da Memória** — conecte função ↔ descrição
- **Adivinhe a Saída** — dado um código C, qual o output?
- **Complete o Código** — preencha as lacunas da implementação
- **Caça-Palavras** — encontre os nomes das funções na grade
- **Referência Rápida** — consulte protótipos, parâmetros e exemplos

## Stack

- Vite + React + TypeScript
- Tailwind CSS v4
- shadcn/ui
- wouter (roteamento)
